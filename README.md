# STRATIGIRAFFIC (SGF)

Repository for the STRATIGIRAFFIC installer packages, demo datasets, and user guide.

This is a public repository under active development. The latest installer is distributed as the GitHub pre-release build dated 2026-03-24.

## Contents

- `AC-demo.xlsx`: example age-constraint table
- `LSS-demo.csv`: example local stratigraphic sequence table
- `Sample table for calibration_demo.csv`: sample table for age calibration
- `Age model-demo.mat`: example output age-model file
- `STRATIGIRAFFIC_UserGuide.pdf`: user guide
- `STRATIGIRAFFIC_UserGuide.docx`: editable user guide source

## Intended Use

This repository is intended to host continuously updated SGF release assets:

- installer packages
- demo input/output files
- user guide documents
- release notes for each published version

## Recommended Release Layout

For future updates, it is recommended to keep:

- root folder: latest stable demo files and core documentation
- `releases/`: archived release notes or version manifests
- GitHub Releases: installer binaries for each published version
